Apache 配置虚拟域名

在本地调试后端项目时候经常会用到虚拟域名的功能；

毕竟项目都放在 WWW 目录下，如果不使用虚拟域名，URL 实在太长，实在不方便调试。

例如：http://localhost/project/public/index.php/sample/test/hello
