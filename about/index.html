陟上晴明

[Zhì Shàng Qíng Míng]
你问我这个名字有没有什么特殊来由，那就点击一下看看？

不知名菜鸡切图仔，从事前端开发7年，主要技术栈为 Vue 全家桶。
准备在博客中记录我的日常学习和工作中遇到的一些问题，希望不要烂尾 23333。

职业生涯早期大部分时间钻研在 CSS 上面，可以说是原地踏步了好多年，直到 2019 年开始接触 SPA 项目后才算真正推开 JavaScript 的大门接触到真正的前端领域。
所以 2019 年的重新开始，对我来说可谓是充满挑战和急速成长的一年，让我从一个萌新蜕变成长为一个称职的前端开发者。

2020年开始因为口罩原因日常娱乐活动开始受限，逐渐开始在开发者社区/社群活跃，在不断的和其他开发者沟通中成长进步。
日常工作生活中你可以在 SegmentFault 和 V2EX 看到我出没的身影。

很多人好奇我的社区ID 【陟上晴明】，想知道有没有什么特殊的来由

早些年有一个《汉字守护者》的活动，参与活动之后会随机分配一个“濒危汉字”，那个时候随机到的就是 【陟】zhì 这个字，寓意也很不错：

登，从低处走向高处，也有晋升和提拔的意思。

当时就在想应该怎么样去守护这个汉字，让更多人去知道和了解他的读音和含义。😂 因为我抽到这个字之后，一下子不知道应该怎么读，所以就想着那就用这个字作为昵称吧。

并且分配到这个字的时候也会给到一首古诗，就觉得搭配到这个字来源的古诗会更有画面一些。所以就用了【陟上晴明】，希望自己能够一路登上云顶，拨开云雾见光明。

扁舟水淼淼，曲岸复长塘。古寺春山上，登楼忆故乡。
云烟横极浦，花木拥回廊。更有思归意，晴明陟上方。
《题山寺》 - 刘商
