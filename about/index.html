陟上晴明

不知名菜鸡切图仔，目前主要技术栈为 Vue 全家桶。
准备在博客中记录我的日常学习和工作中遇到的一些问题，希望不要烂尾 23333。

职业生涯早期大部分时间钻研在 CSS 上面，生涯早期可以说是原地踏步，直到 2019 年初才开始接触 SPA 项目后才算真正推开 JavaScript 的大门，接触到真正的前端领域。
所以 19 年的重新开始，对我来说可谓是充满挑战和急速成长的一年，从一个萌新成长为一个称职的前端开发者。

2020年开始因为口罩原因日常娱乐活动开始受限，逐渐开始在开发者社区/社群活跃，在不断的和其他开发者沟通中成长进步。
日常工作生活中你可以在 SegmentFault 和 V2EX 看到我出没的身影。
