Javascript 滚动侦测导航

今天一个以前的小项目要改，新增一个根据时间线显示不同月份绩效的需求。
没有设计稿，想了下怎么做出来好看。就有了以下这个想法

左侧显示时间线，滑动会显示不同的年份和月份，
右侧显示不同月份的统计数据。

为了显示这个想法不是很蠢，所以需要做的很 酷炫 ，
其实就是交互友好些，左右块都有相应的交互动作。

所以设想的是，点击左侧月份之后右侧会快速滚动到相应的月份，方便用户操作。
当然右侧下滑操作的时候左侧的时间线也会对应的滚动。

那么就要用到滚动侦测了，以前都是用的 UI 框架里边的，所以这次也是直接去翻阅了这个项目所用到的 UI，MuseUI 的文档。

当然里边没有这个组件，不然我也不会些这篇文了

去翻阅了一下 BootStrap 的 Scrollspy 源码，其实就是用到了内容元素的 offsetTop 和滚动条的监听

所以我存储了右侧每月数据部分的 offsetTop ，左侧时间线我没有存储，
因为左侧没有做滚动月份右侧相应的也滑动到对应的月份，只做了点击控制右侧的内容

对，我使用的 Vue 作为框架。

有事要忙了，等空了再补。
