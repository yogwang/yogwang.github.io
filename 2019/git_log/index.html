Git 查看提交日志

git log

基础查看提交日志

git log --oneline

精简模式显示

git log --pretty=oneline

精简模式并显示完整版本号

git log --graph

图形模式显示

git log --stat

显示文件更改列表

git log --author=_name

根据作者筛选

_name：作者名

git log -p file_name

查看某个文件的详细修改   有详细内容

file_name：文件名

git log -L start,end:file_name

查看某个文件某几行范围内的修改记录

start:开始行号

end:结束行号

file_name:文件名

git log --stat commit_id

根本版本号查看提交修改列表

commit_id：提交版本号
