博客改版准备中...

今天突然就想把自己的博客的主题给改了，并且在想如何在 hexo 中使用 vue。

如果可以的话，应该会出一个文章。如果不行的话，还是使用 ejs 或者 pug 来制作(或者使用 VuePress??)
这次可能会多使用几个插件并且会提交到 hexo 官方，并且会考虑 i18n。

但是最近需要进项的项目有 2 个，并且有 2 个项目在测试中，不知道会不会有足够的时间。

先起一个 🐱‍🏍 Flag 吧，发不出去让自己抽时间不偷懒，虽然还有好多未填的坑。
真的是拖了又拖，不过确实去年到今年应为经济管理出现了问题，琐事一直比较多，也很影响心情。

大概率应该又是一个性冷淡风格的主题，不会有很复杂设计在里边，
并且考虑把之前自己的工作案例放进 blog 中展示效果图（链接的话就不放进去了，早期的项目已经有好几年了，早就记不得网址了）

等完成之后也会把现在的这个主题给完善好，也提交上去，这个应该比较快。因为现在这个主题的功能很少。

在 Google 了之后，发现了一个比较蠢的问题，就是 vue 和 hexo 实现的不同，所以在 hexo 中使用 vue 基本不会考虑了，之前的想法有点蠢。

所以要么就是原来的方式，或者使用 VuePress，只是现在还需要了解一下 VuePress。

  • 2020.03.03 现在看来好像 VuePress 没有 Hexo 来的成熟，缺少了我需要的用到的很多插件，虽然可以自己写，但是还是不折腾了。
