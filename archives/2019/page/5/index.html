安装 Git in windows


Git 官网：https://git-scm.com/

设置提交时的 NAME 和 EMAIL：

$ git config --global user.name "nikename"

$ git config --global user.email "example@example.com"

查看设置的 NAME 和 EMAIL：

$ git config user.name

$ git config user.email
